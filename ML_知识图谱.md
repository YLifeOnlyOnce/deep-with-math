## 1. 数学基础

- 1.1 微积分（导数、偏导、梯度、链式法则）
- 1.2 线性代数（向量、矩阵、特征值/特征向量、SVD）
- 1.3 概率统计（条件概率、常见分布、期望方差、贝叶斯、MLE）
- 1.4 最优化（梯度下降、凸优化、约束优化、拉格朗日乘子）
- 1.5 信息论（熵、交叉熵、KL散度，可选强化）

关联到：回归、分类、EM、SVM、PCA、神经网络

### 数学基础推荐学习顺序

微积分 → 线性代数 → 概率统计 → 最优化 → 信息论

### 数学基础与模型映射

- 线性回归：线代 + 微积分 + 最优化
- Logistic 回归：概率统计 + 微积分 + 最优化
- SVM：线代 + 凸优化 + 拉格朗日对偶
- PCA：线代（特征分解/SVD）+ 方差最大化思想
- EM / GMM：概率统计 + 极大似然 + 迭代优化
- 神经网络：链式法则 + 梯度下降 + 概率损失（交叉熵）
